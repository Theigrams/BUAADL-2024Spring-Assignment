# Homework2：CIFAR10 图像分类

## 任务描述

此次作业包含两个实验，分别是：

1. **从头实现数据预处理**：手动实现图像数据的预处理，包括图像的读取、预处理、数据增强。
2. **CIFFAR-10图像分类**：自己实现一个CNN，并使用超参数优化工具自动调参。

## 任务要求

1. Lab1：通过 `test_data_scratch.py` 的测试。
2. Lab2：20个epoch内，在测试集上的准确率达到80%。
